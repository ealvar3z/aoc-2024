Advent of Code 2024
-------------------

Advent of Code solutions:

  - C (C23 standard)
  - C++
  - Go
  - Fortran
  - LuaJIT
  - Perl
  - Plan9 C
  - Plan9 rc
  - Python

Usage:
------
To build all solutions:
	- run the mkfile or Makefile in each dir || the req'd interpreter

Input Files:
------------
Ensure that your input files (e.g., input.txt, sample.txt) are in the same directory as the code.

Dependencies:
-------------
This project requires Plan 9 tools for the `mk` build system:  

	- `plan9port` or `9base`.
